# whatsapp-bot-ryza
WhatsApp bot with multi-device support. Features group management (welcome, promote, demote, kick, add), anti-link protection, and media downloader for TikTok, Instagram, YouTube, MediaFire. Built with Baileys library.

## 📋 About

Ryza Assist is a WhatsApp bot built with Baileys library supporting Multi-Device. Designed for group management and media downloading with a clean and professional architecture.

## ✨ Features

| Category | Features |
|----------|----------|
| **Group Management** | Welcome/Leave messages, Promote/Demote admin, Kick/Add members, Anti-link protection |
| **Media Downloader** | TikTok, Instagram, YouTube (MP3/MP4), MediaFire |
| **Utilities** | Auto-read messages, Plugin system, Multi-device support |
